# [ShowHiddenThings](https://pharaoh2k.github.io/BetterDiscordStuff/?plugin=ShowHiddenThings "ShowHiddenThings") Changelog

### 1.0.0
- Initial public release
- Show member timeout icons in chat without timeout permissions
- Display invites paused tooltip in server list
- Access Mod View context menu option in all servers
- View your own profile in Mod View from context menu
- Display highest role information in mod view
- Automatic update checking with changelog display
- Settings panel with individual toggles for each feature
