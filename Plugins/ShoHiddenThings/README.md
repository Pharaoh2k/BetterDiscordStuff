## ğŸš€ Usage

Once enabled, the plugin automatically reveals hidden elements based on your settings:

### Timeout Icons
- Member timeout indicators will appear in chat messages
- You'll see timeout status even if you don't have timeout permissions
- Note: Viewing the icon doesn't grant actual timeout abilities

### Invites Paused Tooltip
- Hover over servers in your server list to see if invites are paused
- Normally only visible to users with Manage Server permission

### Mod View Features
- Right-click any user to see "Open in Mod View" option
- View detailed moderation information including highest role
- Access mod view for your own profile in any server

## âš ï¸ Important Notes

- This plugin only reveals **visual elements** - it does not grant actual moderator permissions
- You cannot perform moderation actions (timeouts, bans, etc.) without proper permissions
- The plugin is purely informational and does not bypass Discord's permission system
- Some features may not work correctly if Discord updates their internal structure
