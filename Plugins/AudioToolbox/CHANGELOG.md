# [AudioToolbox](https://pharaoh2k.github.io/BetterDiscordStuff/?plugin=AudioToolbox "AudioToolbox") Changelog

### 2.0.1
- Updated meta name and description

### 2.0.0
- Rebranded from AudioDownloader to AudioToolbox
- Added playback speed control (0.5x, 0.75x, 1x, 1.25x, 1.5x, 2x, 3x)
- Added professional EQ presets (Off, Bass Boost, Treble Boost, Voice Enhance, Bass Cut)
- Added basic loop mode for continuous audio playback
- Added A/B loop mode for precise section repetition
- Added queue mode for automatic sequential playback of audio messages
- Added inline controls (loop button and three-dot menu) on audio messages
- Added batch download feature for all cached audio in a channel
- Added auto-download capability for specific channels
- Added download history tracking (up to 200 entries)
- Added format conversion support (WAV and M4A/AAC)
- Added custom filename templates with multiple tokens
- Added "Show 'Open in Folder' Toast" option with clickable folder link
- Improved context menu organization with submenus
- Improved download progress tracking for large files
- Improved file deduplication with automatic numbering
- Improved settings panel with history management and changelog viewer
- Multiple UI/UX improvements for better Discord theme integration
- Enhanced error handling and user feedback throughout
- Comprehensive automatic update system with changelog display

### 1.0.0
- Initial public release as AudioDownloader
- Basic audio download functionality
- Support for voice messages and audio attachments
- Download mode selection (direct, save-as, browser)
- Custom download folder configuration
- Context menu integration
