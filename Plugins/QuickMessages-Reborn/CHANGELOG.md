# [QuickMessages-Reborn](https://pharaoh2k.github.io/BetterDiscordStuff/?plugin=QuickMessages-Reborn "QuickMessages-Reborn") Changelog

### 1.0.0
- Initial public release
- Save messages for quick reuse via right-click context menu
- Organize messages into custom categories
- Uncategorized messages support
- Save selected text or entire messages
- Shift+Click to delete messages
- Shift+Click to delete entire categories
- Insert saved messages into chat instantly
- Settings panel to manage all saved items
- Delete all data option in settings
- Automatic update checking every 24 hours
- Changelog display on updates
- Discord theme integration
- Support for Discord formatting and emojis in messages
- Text truncation for long messages (>50 chars) in menu display
- Modern BdApi implementation (no ZeresPluginLibrary dependency)
- Inspired by QWERTxD's original "QuickMessages" plugin
