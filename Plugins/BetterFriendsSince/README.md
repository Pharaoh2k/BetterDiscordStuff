# BetterFriendsSince

A BetterDiscord plugin that displays when you and another user became friends directly in their profile and their sidebar.

![Version](https://img.shields.io/badge/version-1.3.7-blue.svg)
![Status](https://img.shields.io/badge/status-active-success.svg)
![Discord](https://img.shields.io/badge/discord-BetterDiscord-7289DA.svg)

## 📖 Description

BetterFriendsSince enhances Discord's user profile modal by adding a "Friends since" section that shows the exact date when you became friends with someone. The date is displayed in a clean, native-looking format that seamlessly integrates with Discord's UI.

## ✨ Features

- **📅 Friendship Date Display**: See exactly when you became friends with someone
- **🌍 Multi-Language Support**: Automatically adapts to your Discord language (24+ languages supported)
- **🎨 Native UI Integration**: Uses Discord's own components for a seamless look
- **⚡ Reactive Updates**: Automatically updates when friendship status changes
- **🔄 Localized Dates**: Date format adapts to your locale preferences
- **🪶 Lightweight**: Minimal performance impact on Discord

## 📦 Installation

1. Download [BetterFriendsSince.plugin.js](https://raw.githubusercontent.com/Pharaoh2k/BetterDiscordStuff/main/Plugins/BetterFriendsSince/BetterFriendsSince.plugin.js)
2. Place the file in your BetterDiscord plugins folder:
   - Windows: `%appdata%\BetterDiscord\plugins`
   - macOS: `~/Library/Application Support/BetterDiscord/plugins`
   - Linux: `~/.config/BetterDiscord/plugins`
3. Enable the plugin in Discord Settings → Plugins

## 🚀 Usage

1. Click on any user to open their profile
2. If you're friends with them, a "Friends since" section will appear in their profile
3. The date shows when your friendship began

## 🔧 Troubleshooting

### "Friends since" not showing?
- Ensure you are actually friends with the user
- Reload Discord with Ctrl+R (Cmd+R on Mac)
- Check that the plugin is enabled in settings

### Plugin failed to initialize?
- Open Discord Developer Console (Ctrl+Shift+I) to check for errors
- Ensure you're using the latest version of BetterDiscord
- Try disabling other plugins to check for conflicts
